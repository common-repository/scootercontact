ENG : 

ULTRA Light plugin allowing to display a light and secure contact form with shortcode.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Insert the contact form where you want with the following shortcode [scootercontact], either directly in your articles / pages, or directly in your theme (in footer.php for example)


.::: EXPLICATIONS EN FRANCAIS :::.

1. Téléchargez les fichiers de l'extension dans le répertoire `/wp-content/plugins/plugin-name`, ou installez directement le plugin via l'écran des plugins WordPress.
2. Activez le plugin via l'écran "Extensions" de WordPress
3. Insérez le formulaire de contact où vous voulez avec le shortcode suivant [scootercontact], soit directement dans vos articles/pages, soit directement dans votre theme (en footer.php par exemple)
